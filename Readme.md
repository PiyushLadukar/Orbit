# 🛰️ Orbit

> Your Personal AI Career Agent

Orbit automatically searches for internships and full-time jobs, matches them with your resume, generates tailored resumes and cover letters, and prepares applications using a completely free stack.

## Features

- Find jobs from multiple sources
- AI-powered resume matching
- ATS optimized resumes
- AI-generated cover letters
- Gmail draft generation
- Daily job summary
- 100% Free

## Tech Stack

- n8n
- Google Gemini
- Supabase
- Google Drive
- Gmail
- Oracle Cloud
